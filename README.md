
## Description
Project about company Makadeshi Events. This project is for educational purposes at the request BlueEdtech.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
